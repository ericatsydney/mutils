# Agents

This directory contains project-specific agent profiles.

## Available Agents

- `developer`: implementation, debugging, refactoring, testing, and code review.
- `ui-designer`: UI design, interaction design, frontend polish, and design-system guidance.
- `marketing-research`: market research, audience research, competitor analysis, and positioning.

Agent metadata lives in `.agents/settings.json`. Each agent folder contains an `AGENTS.md` file with role-specific operating instructions.
